Agro-Meteorological Analytics: Weather-Driven Crop Stress & Insights

# Project Overview
This repository contains an end-to-end data analytics framework that pairs real-time weather observation streams with specific crop metadata. By evaluating environmental factors against biological limits, the project helps track crop health, model localized water demand indices, and trigger urgent irrigation alerts.

The implementation is broken down into two core project files, detailed below.

---

# Repository File Breakdown

### 1. `weather_crop_analysis.sql`
This file contains the backend relational database script and data engineering pipeline for the project. Written in optimized SQL, it joins real-time weather streams, city mappings, and biological crop thresholds to run four key analytical models:
* **Unified Observational Layer:** Joins daily meteorological figures directly with targeted regional crops.
* **Crop Stress Classification Engine:** Uses conditional logic to dynamically categorize environmental strain into *Heat Stress*, *Cold Stress*, or *Optimal* by matching actual temperatures against crop-specific biological limits.
* **Water Demand Modeling:** Computes a specialized mathematical Water Demand Index for each crop type based on average ambient temperatures, humidity drops, and unique crop growth coefficients ($K_c$).
* **Automated Urgent Irrigation Alerts:** An anomaly detection query that isolates specific cities facing zero precipitation alongside surging water demand, outputting a prioritized list for immediate agricultural intervention.

### 2. Google Sheets: Weather Crop Analysis Visuals (Screenshots)
This section showcases the reporting and business intelligence visualization layer of the project. Built using Google Sheets, these interactive, stakeholder-facing dashboards ingest the structured outputs generated by the SQL queries. The included screenshots demonstrate:

Operational Summaries: High-level executive views aggregating weather anomalies and crop health metrics.

Analytical Charts & Data Fields: Visual trend charts mapping out water depletion spikes across different regions and tracking localized crop stress breakdowns.

Pivot Tables: Dynamic, scannable data grids that allow users to filter insights by specific cities or primary crop types.
